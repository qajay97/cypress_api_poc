# Cypress API Testing Sample Project

This is a sample project demonstrating how to perform API testing using Cypress. Please note that this is sample only and won't work as-is since it lacks actual API endpoints and credentials.

## Prerequisites

- Node.js and npm installed on your machine.

## Project Structure

- cypress/tests: Contains the test files where actual test cases are defined.
- cypress/support: Contains various supporting files and custom commands.
- commands.js: Defines custom Cypress commands.
- support/models: Holds TypeScript interfaces for API response models.
- support/services: Houses the logic for making API requests.
- constants: Stores constants used throughout the tests.
- cypress/fixtures: Contains JSON files with data for testing.
